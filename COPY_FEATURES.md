# תכונות העתקה - Logcat Viewer

## סקירה כללית

התוכנה כוללת מערכת העתקה מתקדמת המאפשרת העתקה גמישה של לוגים בדרכים שונות.

## תכונות העתקה

### 1. העתקת הודעה בודדת
**איך להשתמש:**
- לחץ לחיצה ימנית (right-click) על שורת לוג
- בחר "העתק הודעה"
- רק תוכן ההודעה יועתק ללוח

**דוגמה:**
```
Application started successfully
```

### 2. העתקת שורה מלאה
**איך להשתמש:**
- לחץ לחיצה ימנית (right-click) על שורת לוג
- בחר "העתק שורה מלאה"
- כל פרטי הלוג יועתקו בפורמט מלא

**דוגמה:**
```
12-25 10:30:45.123 1234/5678 I/MainActivity: Application started successfully
```

### 3. בחירה והעתקה של מספר שורות
**איך להשתמש:**
1. לחץ על שורת לוג ראשונה (היא תסומן בכחול)
2. החזק Ctrl ולחץ על שורות נוספות לבחירה מרובה
3. לחץ לחיצה ימנית על אחת השורות המסומנות
4. בחר "העתק X שורות" (X = מספר השורות שנבחרו)
5. כל השורות יועתקו בפורמט מלא, מופרדות בשורות חדשות

**דוגמה:**
```
12-25 10:30:45.123 1234/5678 I/MainActivity: Application started successfully
12-25 10:30:45.456 1234/5678 D/NetworkManager: Connecting to server
12-25 10:30:45.789 1234/5678 I/NetworkManager: Connection established
```

### 4. ביטול בחירה
**איך להשתמש:**
- לחץ לחיצה ימנית על שורות מסומנות
- בחר "בטל בחירה"
- כל הסימונים יוסרו

## עיצוב ואינדיקציות חזותיות

### צבעים
- **רגיל**: רקע אפור כהה (#252525)
- **Hover**: רקע אפור בהיר יותר (#2D2D2D)
- **נבחר**: רקע כחול כהה (#1E3A5F)

### סדר עמודות (LTR)
התצוגה מעוצבת בסדר משמאל לימין (LTR) כיון שהתוכן באנגלית:

```
Timestamp → PID/TID → Level → Tag → Message
```

**דוגמה:**
```
12-25 10:30:45.123  1234/5678  I  MainActivity  Application started successfully
```

## טיפים לשימוש

1. **בחירה מהירה**: לחץ על שורה אחת ללא Ctrl כדי לבחור רק אותה
2. **בחירה מרובה**: החזק Ctrl ולחץ על שורות נוספות
3. **העתקה מהירה**: לחיצה ימנית ישירות על שורה לא מסומנת תציג תפריט העתקה בודדת
4. **העתקה מרובה**: לחיצה ימנית על שורה מסומנת תציג תפריט העתקה מרובה

## יתרונות

- **גמישות**: בחר בדיוק מה להעתיק - הודעה בלבד או שורה מלאה
- **יעילות**: העתק מספר שורות בבת אחת
- **נוחות**: תפריט הקשר אינטואיטיבי וקל לשימוש
- **חזותי**: אינדיקציות ברורות למצב הבחירה

## הערות טכניות

- השימוש ב-AWT Clipboard מבטיח תאימות מלאה עם Windows
- הטקסט מועתק בפורמט טקסט רגיל (plain text)
- תומך בהעתקה של מספר בלתי מוגבל של שורות
- הביצועים אופטימליים גם עם מספר רב של שורות נבחרות
