# מדריך שימוש - בחירה והעתקה של לוגים

## 🖱️ בחירת שורות

### בחירה בודדת
- **לחץ** על שורה → השורה נבחרת (מסומנת בכחול)

### בחירה מרובה בגרירה
1. **לחץ** על שורה ראשונה
2. **החזק** את כפתור העכבר
3. **גרור** למעלה או למטה
4. כל השורות בטווח יסומנו בכחול
5. **שחרר** את הכפתור

### בחירה מרובה עם Ctrl
1. **לחץ** על שורה ראשונה → נבחרת
2. **Ctrl + לחץ** על שורה נוספת → מתווספת לבחירה
3. **Ctrl + לחץ** על שורה נבחרת → מוסרת מהבחירה
4. ניתן לבחור שורות לא רציפות

### ביטול בחירה
- **לחץ** על שורה לא נבחרת (ללא Ctrl) → מבטל את כל הבחירה
- או: לחיצה ימנית → "בטל בחירה"

## 📋 העתקה ללוח

### העתקת שורה בודדת
1. **לחיצה ימנית** על שורה
2. בחר אחת מהאופציות:
   - **"העתק הודעה"** - רק טקסט ההודעה
   - **"העתק שורה מלאה"** - timestamp + pid/tid + level + tag + message

### העתקת שורות מרובות
1. **בחר** מספר שורות (בגרירה או עם Ctrl)
2. **לחיצה ימנית** על אחת מהשורות הנבחרות
3. בחר **"העתק X שורות"**
4. כל השורות מועתקות ללוח, כל אחת בשורה נפרדת

### פורמט השורה המלאה
```
01-15 10:23:45.123 1234/5678 I/MyTag: This is a log message
```

## 🎨 אינדיקטורים ויזואליים

### צבעים
- **אפור כהה** (רגיל) - שורה לא נבחרת
- **אפור בהיר** (hover) - עכבר מעל השורה
- **כחול כהה** - שורה נבחרת

### רמות לוג
- **V** (Verbose) - אפור
- **D** (Debug) - כחול
- **I** (Info) - ירוק
- **W** (Warning) - צהוב
- **E** (Error) - אדום
- **A** (Assert) - סגול

## ⌨️ קיצורי מקלדת

| פעולה | קיצור |
|-------|-------|
| בחירה מרובה | Ctrl + Click |
| הדבקה | Ctrl + V (במחברת/עורך טקסט) |

## 💡 טיפים

### טיפ 1: בחירה מהירה
- גרור מהר מאוד - הבחירה תעקוב בצורה חלקה
- אין צורך לחכות - הקאשינג מטפל בכל

### טיפ 2: בחירה של טווח גדול
1. לחץ על השורה הראשונה
2. גלול למטה (עם גלגלת העכבר)
3. Shift + Click על השורה האחרונה
   - (אם זה לא עובד, השתמש בגרירה)

### טיפ 3: העתקה מהירה
- אם אתה צריך רק את ההודעה, לא את כל הפרטים
- השתמש ב"העתק הודעה" במקום "העתק שורה מלאה"

### טיפ 4: סינון לפני העתקה
1. השתמש בפילטרים (חיפוש, תג, רמה)
2. בחר את כל השורות הרלוונטיות
3. העתק
4. תקבל רק את מה שאתה צריך

## 🔍 דוגמאות שימוש

### דוגמה 1: מציאת שגיאות
```
1. סנן לפי רמה: E (Error)
2. חפש: "NullPointerException"
3. בחר את כל השורות הרלוונטיות
4. העתק
5. הדבק בדוח באג
```

### דוגמה 2: ניתוח flow
```
1. סנן לפי תג: "MyActivity"
2. בחר את כל השורות מרגע הפתיחה
3. העתק שורות מלאות
4. נתח את הזמנים והסדר
```

### דוגמה 3: שיתוף עם צוות
```
1. מצא את הבעיה
2. בחר 5-10 שורות סביב הבעיה
3. העתק
4. שלח בצ'אט/מייל
```

## ⚠️ הערות חשובות

### ביצועים
- הבחירה עובדת חלק גם עם אלפי שורות
- הקאשינג מטפל בכל ברקע
- אין lag או המתנה

### מגבלות
- ניתן לבחור רק שורות שנטענו (בחלון הנראה + 1000 שורות)
- אם אתה צריך יותר, השתמש ב"ייצוא לוגים" (💾)

### תאימות
- העתקה עובדת עם כל אפליקציה (מחברת, Word, IDE, וכו')
- הפורמט הוא טקסט רגיל
- שורות מופרדות ב-newline

## 🆘 פתרון בעיות

### הבחירה לא עובדת
- ודא שאתה לוחץ ומחזיק את הכפתור
- נסה לגרור לאט יותר
- בדוק שיש לוגים בחלון

### תפריט ההקשר לא מופיע
- ודא שאתה לוחץ לחיצה ימנית
- בדוק שהשורה נבחרת (כחולה)
- נסה שוב

### ההעתקה לא עובדת
- בדוק שבחרת שורות
- נסה Ctrl+V במחברת
- ודא שהתפריט נסגר אחרי הבחירה

## 📞 תמיכה

אם נתקלת בבעיה:
1. בדוק את הקונסול לשגיאות
2. נסה לסגור ולפתוח מחדש
3. דווח על הבעיה עם צילום מסך
