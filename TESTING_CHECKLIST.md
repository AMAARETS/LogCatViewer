# רשימת בדיקות לשינויים החדשים

## בדיקות לתיקון בחירת שורות מרובות

### בחירה בגרירה (Drag Selection)
- [ ] לחץ על שורה אחת - אמורה להיבחר
- [ ] החזק את הכפתור ושלוף למטה - כל השורות בטווח אמורות להיבחר
- [ ] החזק את הכפתור ושלוף למעלה - כל השורות בטווח אמורות להיבחר
- [ ] שלוף מהר מאוד - הבחירה אמורה לעקוב בצורה חלקה
- [ ] שלוף מעבר לשורות שנטענו - אמורה להיות בחירה רציפה

### בחירה עם Ctrl
- [ ] Ctrl+Click על שורה - אמורה להתווסף לבחירה
- [ ] Ctrl+Click על שורה נבחרת - אמורה להוסר מהבחירה
- [ ] Ctrl+Click על מספר שורות לא רציפות - כולן אמורות להיבחר

### ויזואליזציה
- [ ] שורות נבחרות מוצגות בצבע כחול כהה (0xFF1E3A5F)
- [ ] Hover על שורה לא נבחרת מציג צבע אפור בהיר
- [ ] הבחירה נשארת גלויה גם כשהעכבר לא מעליה

## בדיקות להעתקה

### העתקה של שורה בודדת
- [ ] לחיצה ימנית על שורה -> "העתק הודעה" -> מועתק רק הטקסט של ההודעה
- [ ] לחיצה ימנית על שורה -> "העתק שורה מלאה" -> מועתק timestamp + pid/tid + level + tag + message

### העתקה של שורות מרובות
- [ ] בחר מספר שורות בגרירה
- [ ] לחיצה ימנית על אחת מהשורות הנבחרות
- [ ] בחר "העתק שורות מסומנות"
- [ ] **הטקסט אמור להיות מועתק מיד ללוח ללא תפריט נוסף**
- [ ] הבחירה אמורה להתבטל אחרי ההעתקה
- [ ] הדבק (Ctrl+V) במחברת - אמור להראות את כל השורות הנבחרות, כל אחת בשורה נפרדת

### בדיקת תוכן מועתק
```
דוגמה לפורמט צפוי של שורה מלאה:
01-15 10:23:45.123 1234/5678 I/MyTag: This is a log message
```
