# Logcat Viewer - מציג לוגים לאנדרואיד

תוכנה מודרנית לצפייה בלוגים של מכשירי אנדרואיד מחוברים, נבנתה בקוטלין עם Compose Multiplatform.

## תכונות

- ✅ תמיכה בכל הפלטפורמות (Windows, macOS, Linux)
- 🎨 עיצוב מודרני ואפל
- 🚀 **ADB מובנה** - התוכנה מורידה ומריצה ADB אוטומטית!
- 🔍 חיפוש וסינון מתקדם
- 🏷️ סינון לפי רמת לוג (V/D/I/W/E/A)
- 🔖 סינון לפי תגים
- 💾 ייצוא לוגים לקובץ
- 🔄 גלילה אוטומטית
- 📱 זיהוי אוטומטי של מכשירים מחוברים
- ⚡ ביצועים גבוהים עם **אחסון בלתי מוגבל** ב-SQLite
- 💾 **טעינה וירטואלית חכמה** - רק מה שנראה על המסך נטען לזיכרון
- 🗄️ **מסד נתונים מקומי** - כל הלוגים נשמרים ב-DB מהיר
- 🔌 ללא צורך בהתקנת Android SDK!

## דרישות מקדימות

- JDK 8 או גבוה יותר
- מכשיר אנדרואיד עם USB Debugging מופעל

**זהו!** התוכנה תוריד ותתקין את ADB אוטומטית בהרצה הראשונה.

## התקנה והרצה

### הרצה במצב פיתוח

```bash
./gradlew run
```

### בניית קובץ הפצה

**Windows:**
```bash
./gradlew packageMsi
```

**macOS:**
```bash
./gradlew packageDmg
```

**Linux:**
```bash
./gradlew packageDeb
```

הקבצים יווצרו בתיקייה: `build/compose/binaries/main/`

## שימוש

1. הרץ את התוכנה (בהרצה הראשונה היא תוריד ADB אוטומטית)
2. חבר מכשיר אנדרואיד עם USB Debugging מופעל
3. בחר מכשיר מהרשימה הנפתחת
4. לחץ על כפתור ההפעלה (▶️) להתחלת קבלת לוגים
5. השתמש בסינונים לחיפוש ולסינון לוגים
6. לחץ על כפתור השמירה (💾) לייצוא לוגים לקובץ

## טכנולוגיות

- **Kotlin** - שפת התכנות
- **Compose Multiplatform** - UI framework מודרני
- **JADB** - Pure Java implementation של פרוטוקול ADB
- **EmbeddedAdb** - מערכת ניהול ADB אוטומטית
- **Coroutines** - עיבוד אסינכרוני
- **ADB (Google)** - מוטמע ומופעל אוטומטית

## איך זה עובד?

התוכנה משתמשת ב-**JADB** (Pure Java ADB) + **ADB מוטמע**!

### ארכיטקטורה:

```
התוכנה (Kotlin/Compose)
    ↓
EmbeddedAdb - מנהל ADB אוטומטי
    ↓ (מוריד ומפעיל אם נדרש)
ADB Server (daemon ברקע)
    ↓
JADB (Pure Java) - תקשורת דרך socket
    ↓
USB Driver
    ↓
מכשיר Android
```

### מה זה אומר?

**EmbeddedAdb:**
- ✅ בודק אם ADB קיים ב-`~/.logcat-viewer/`
- ✅ אם לא - מוריד אוטומטית מ-Google
- ✅ מפעיל את ADB server
- ✅ מוודא שהוא רץ

**JADB:**
- ✅ **Pure Java** - ללא JNI או native code
- ✅ מתחבר ל-ADB server דרך socket (localhost:5037)
- ✅ שולח פקודות בפרוטוקול ADB
- ✅ מקבל לוגים בזמן אמת
